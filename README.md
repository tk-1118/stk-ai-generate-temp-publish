# @hhzy/ai-generate-template

一个可直接生成前端项目的模板脚手架。

## 环境要求
- Node >= 18

## 安装与使用

无需全局安装，推荐使用 npx：

```bash
npx @hhzy/ai-generate-template my-project
cd my-project
npm run dev
```

或全局安装后使用：

```bash
npm i -g @hhzy/ai-generate-template
ai-generate-template my-project
```

## 模板说明
- 打包工具：Vite
- 框架：Vue 3
- UI：Element Plus、Avue
